# Step_verilog

## 项目介绍
  这是一个Argon 编写的小脚丫FPGA 板子的Demo例程



| 文件名      | 说明             |
| ----------- | ---------------- |
| 1.led_pwm.v | led灯闪烁 |
| 2.1  led_key(with module).v | 按键测试 |
| 2.2 | 按键消抖 |
| 3.pwm(with  module) | 模块形式的PWM输出 |
| 4.Uart | 串口收发 |
| 5.DS18B20   | DS18B20 串口输出 |
| 6.SPI       | SPI OLED的使用   |
| 7.clock_div | 分频器的使用     |
| ending | 完整功能程序 |
| python_uart | 上位机程序 |
